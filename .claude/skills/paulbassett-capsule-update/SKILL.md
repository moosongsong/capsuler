# 폴바셋(Paul Bassett) 캡슐·패키지 업데이트

네이버 브랜드스토어 API(`scripts/paulbassett-fetch.sh`)로 캡슐커피 카테고리를 가져와
`src/data/capsules/paulbassett.json`(단품, id 6000번대) · `src/data/packages/paulbassett.json`(패키지, id 7000번대)을 최신화한다.
폴바셋 캡슐은 전부 **네스프레소 오리지널 호환**(`compat: original`), 브랜드는 `폴바셋`.

## ⚠️ API의 한계와 관문
- **관문**: `User-Agent` + `Referer` 헤더가 **둘 다** 있어야 200. 없으면 429(봇 차단). `pageSize`는 80 이하(100이면 빈 결과).
- **강도(intensity)·향미(aromatics) 미제공** — 이름·가격·이미지·구매URL만 있다.
  - 신규 단품은 **강도 0·향미 빈값·영문명 자동 생성**으로 추가되고 `[보완 필요]`로 플래그.
  - 실제 강도·향미·영문명·산미/바디/쓴맛은 **폴바셋 공식 상세(제품 설명)** 를 보고 사람이 채운다.
  - 기존 단품의 큐레이션 값(강도/향미/영문명/설명 등)은 절대 덮어쓰지 않는다.

## 상품 분류 (이름: `폴바셋 네스프레소 호환 캡슐커피 {맛}, 10개입, {N}개`)
- **단품**: 특정 맛 + `1개`입 → 캡슐(개당가 = 할인가 ÷ 10)
- **패키지**: `N종 혼합` → 세트(구성 = 현재 단품 전체, 수량 = 총캡슐 ÷ 종수)
- **제외**: 특정 맛 + `2개` 이상(같은 맛 벌크 묶음) — 어소트먼트가 아니라 모델링하지 않는다.

## 정책
- **신규 단품/패키지** → 추가. 가격·이미지·구매URL은 실제값.
- **사라진 항목** → `isEnabled: false`(소프트 단종). **다시 전시** → 해제.
- **기존 단품** → 가격·구매URL·이미지만 갱신, 큐레이션 값 보존.
- **기존 패키지** → 가격·구매URL·이미지·구성(items) 갱신, **영문명(name) 보존**. 구성 비교는 순서 무관.
- 브랜드 `폴바셋`만 대상. 다른 브랜드는 건드리지 않는다.

## 실행 절차
1. **미리보기**: `node scripts/update-paulbassett.mjs --dry-run` → 리포트 검토.
   - ⚠️ 안전장치: 상품 0개면 조회 실패(429/pageSize 초과)로 보고 중단한다.
2. **반영**: `node scripts/update-paulbassett.mjs`
3. **신규 단품 보완**(`[보완 필요]`): 폴바셋 공식 상세(제품 이미지의 맛 설명)를 보고 `name`(영문)·`notes`·`desc`를 채운다. 새 향미가 필요하면 `src/data/notes.json`에 추가.
   - **강도·산미/바디/쓴맛**: 폴바셋은 공식 수치를 공개하지 않는다. 맛 설명 기반으로 **근사치를 추정**해 넣되(예: 다크·카카오·쌉싸름→쓴맛·강도↑, 베리·상큼→산미↑·바디↓), 해당 캡슐의 `desc` 끝에 **"(강도·산미·바디·쓴맛은 공식 수치가 없어 향미 기반 추정치입니다.)"** 안내를 덧붙인다.
4. **신규 패키지 영문명 검토**: `[영문명 검토]` 항목을 자연스러운 세트명으로 다듬는다.
5. **검증**: `npm run typecheck`(필수), 필요시 preview.
6. **보고**: (단품) 추가/단종/가격/보완필요 + (패키지) 신규/단종/변경 요약 전달.

## 커밋
- **요청 없이 커밋·푸시하지 않는다.**

## 참고
- 단품 가격은 `할인가(benefitsView.discountedSalePrice) ÷ 10`(개당), 패키지 가격은 할인가 전체.
- 패키지 구성은 "N종 혼합 = 현재 단품 전체"라는 도메인 지식으로 채운다(API가 구성을 주지 않음). 종수 ≠ 단품 수면 스크립트가 경고하니 확인한다.
- 이미지 필드는 `representativeImageUrl`, 구매URL은 `https://brand.naver.com/baristapaulbassett/products/{id}`.
- `data.ts`가 `paulbassett.json`(캡슐)·`packages/paulbassett.json`을 다른 브랜드와 합쳐 노출한다. 신규 브랜드 추가 시 `data.ts` import와 `i18n`의 `BRAND_LABELS`도 확장한다.
